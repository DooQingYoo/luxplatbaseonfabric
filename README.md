# 基于fabric区块链的奢侈品供应链平台
2020届曲阜师范大学毕业设计  
基于fabric 1.4.1  
## 区块链
NetWork文件夹
### 启动
NetWork文件夹
```bash
# 启动容器
./bcnet.sh up
# 创建通道
./bcnet.sh channel
# 节点加入
./bcnet.sh join
# 安装链码
./bcnet.sh install
# 实例化链码
./bcnet.sh instantiate
```
### 关闭
```bash
./bcnet.sh down
```
